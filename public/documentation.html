myFlix API

A REST API for an application called “myFlix” that interacts with a database that stores data about different movies.


A server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON object holding data about all the movies
Get data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title /movies/[ID] GET None A JSON object holding data about a single movie
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genres/[genre] GET None A JSON object holding data about a genre
Return data about a director (bio, birth year, death year) by name /movies/directors/[director] GET None A JSON object holding data about a single director
Add new user /users POST A JSON object holding data about the student to add, structured like:
{name: "Laina Marciano", username: "lmmarciano", password:"encrypted"}		 A JSON object holding data about the student that was added, including an ID:
{id: 5, name: "Laina Marciano", username: "lmmarciano", password:"encrypted"}
Update user info (username) /users/[ID]/info PUT None A text message indicating user's info was updated
Add movie to a user's favorites /users/[ID]/favorites PUT None A text message that the movie was added to the user's favorites.
Remove a movie by ID /users/[ID]/favorites DELETE None A text message that a movie has been removed from favorites
Allow users to deregister /users/[ID]/deactivate DELETE None A text that a user email has been removed
